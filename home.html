Important FAQ ?

What is the Document Object Model(DOM)?

The DOM is a structure for manipulating HTML and XML documents.

How do you select an element from the DOM

You can select DOM elements using `getElementById`, `querySelector`, or similar methods.

What is event delegation in the context of the DOm,and why is it useful?

Event delegation in the DOM is a technique where a single event listener is added to a parent element to manage events from multiple child elements, making it more efficient for handling dynamic content.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using `setAttribute`, `getAttribute`, or by directly modifying properties, and manipulate styles using the `style` property.
